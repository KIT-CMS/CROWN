auto FilterID(int index)¶

Function to filter the Tau ID in NanoAOD. The disciminator output is stored as a bitmask. In order to filter based on a given WP, a bitwise comparison between the given ID and the filtered index is performed. Return true if the working point is passed by a tau.

Parameters

index – The bitmask index to be used for comparion

Returns

a lambda function to be used in RDF Define
