Crown
=======

The C++-based ROot Workflow for N-tuples (CROWN)  framework is a fast way, of converting CMS NanoAOD samples into analysis N-tuples.


Documentation
--------------

The documentation can be found at https://crown.readthedocs.io/en/latest/.
